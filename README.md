# Paypal-clone
## Project Overview
Paypal-clone is a web application that mimics the functionality of PayPal, allowing users to send money, pay online, or set up a merchant account.
## Key Features
- User authentication and authorization
- Payment processing
- Merchant account setup
## Tech Stack
- HTML
- CSS
- JavaScript
## Installation
To install the project, clone the repository and open the index.html file in a web browser.
## Usage
To use the project, simply navigate to the index.html file and interact with the application as you would with PayPal.
## Environment Variables
No environment variables are required to run the project.
